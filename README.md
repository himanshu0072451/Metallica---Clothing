# 🖤 Metalica Store — Modern Clothing Website

A modern, interactive clothing website built with performance, aesthetics, and user experience in mind.

Designed to transform a local clothing shop into a premium digital experience.

---

## Live Concept

> A cinematic, minimal, and interactive fashion website inspired by modern brands like Zara, COS, and Fear of God.

---

## Features

- Smooth animations (GSAP + Framer Motion)
- Interactive UI with hover effects & micro-interactions
- Modern glass + depth-based design
- Scalable cart system (in progress)
- Fully responsive layout
- Cinematic hero section with loading + text reveal
- Clean and modular architecture (TypeScript)

---

## Tech Stack

- React (Vite)
- TypeScript
- Tailwind CSS
- GSAP
- Framer Motion

---

## Sections

- Hero (cinematic entry experience)
- Featured Collection
- Product Showcase
- Horizontal Scroll Section
- Brand Story
- Instagram-style Grid
- Footer

---

## Goal

This project focuses on:

- Making a **local clothing shop feel like a premium brand**
- Delivering a **smooth, interactive user experience**
- Building a **scalable frontend system ready for e-commerce integration**
